## Prerequisites
* GCC compiler
* Ncurses library (`libncurses5-dev`)


## INSTALLATION
 LINUX:
 sudo apt-get install libncurses5-dev libncursesw5-dev

 MACOS:
 brew install ncurses


##INSTALLATION AND BUILD
git clone https://github.com/yourusername/tetris.git
cd tetris


## Build & Run
make
./tetris
